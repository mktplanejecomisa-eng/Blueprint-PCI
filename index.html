Passo 1 — Baixe o JSON

Clique abaixo para baixar o arquivo respostas-cliente.json com todas as respostas estruturadas.

Passo 2 — Envie ao repositório

Com o arquivo baixado, use um dos métodos abaixo para salvar no GitHub:

Via interface web (mais fácil)
  1. Acesse seu repositório no github.com
  2. Clique em Add file → Upload files
  3. Arraste o arquivo respostas-cliente.json
  4. Escreva a mensagem do commit e clique em Commit changes
Via terminal (Git)
# mova o arquivo para a pasta do repositório
mv ~/Downloads/respostas-cliente.json ./clientes/
git add respostas-cliente.json
git commit -m "add: respostas cliente — Casa de Cuidados"
git push origin main
Passo 3 — Claude Code (automação)

Com o Claude Code instalado, você pode automatizar o envio diretamente pelo terminal sem sair do fluxo:

# dentro da pasta do projeto
claude "commita o arquivo respostas-cliente.json
       na pasta /clientes do repositório planeje-com-isa"

Instale o Claude Code em claude.ai/code para usar esse fluxo.
